####react-native 学习

环境配置

1.安装Homebrew : /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
2.安装watchman : brew install watchman
4.安装flow : brew install flow
5.安装react-native-cli : npm install -g react-native-cli
6.运行环境:
ios:Xcode 7.0 android:
1.Android Studio(记得安装jdk,Android jdk)
2.Genymotion(VirtualBox依赖)

开始第一个react-native项目

react-native init AHelloWorld
cd AHelloWorld
react-native run-ios

ps:android需要运行在Android Studio里面
