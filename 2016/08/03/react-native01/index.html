react-native初尝试

Wednesday, August 3rd 2016

React Native是一个跨平台移动APP开发框架，着力于提高多平台开发的开发效率，使你能够在JavaScript 和react的基础上获得完全一致的开发体验，构建世界一流的原生app

环境配置

  1. 安装Homebrew : /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
  2. 安装watchman : brew install watchman
  3. 安装flow : brew install flow
  4. 安装react-native-cli : npm install -g react-native-cli
  5. 运行环境:
  • ios:Xcode 7.0
  • android:
    1.Android Studio(记得安装jdk,Android jdk)
    2.Genymotion(VirtualBox依赖)

开始第一个react-native项目

  • react-native init AHelloWorld
  • cd AHelloWorld
  • react-native run-ios
    ps:android需要运行在Android Studio里面
